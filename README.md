## 2D functions you can try

* `Math.pow(x, 4) - 2 * Math.pow(x, 2)`

* `Math.sin(x/2) * Math.cos(y/2)`
* `Math.sin(Math.sqrt(x*x + y*y)) / 2`
* `Math.sqrt(x*x + y*y)`
