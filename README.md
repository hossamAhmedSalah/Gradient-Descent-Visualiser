## 2D functions you can try 
* `Math.pow(x, 2)`
* `Math.pow(x, 4) - 2 * Math.pow(x, 2)`
* `Math.cos(x) + 0.1 * Math.pow(x, 2)`
* `10 + Math.pow(x, 2) - 10 * Math.cos(2 * Math.PI * x)`

## 3D functions you can try
* `x*x + y*y`
* `Math.sin(x/2) * Math.cos(y/2)`
* `Math.sin(Math.sqrt(x*x + y*y)) / 2`
* `Math.sqrt(x*x + y*y)`
